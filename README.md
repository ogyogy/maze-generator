# maze-generator

## Usage

```bash
go run .
```

## Game screen

Mazes are randomly generated.

```
#######
#.#...#
#.#@#.#
#...#.#
#.###.#
#.#G..#
#######
move (hjkl), quit (q):
```

| Character | Description |
| --------- | ----------- |
| #         | Wall        |
| .         | Path        |
| @         | Player      |
| G         | Goal        |

## Command

| Key | Action     |
| --- | ---------- |
| h   | Move left  |
| j   | Move down  |
| k   | Move up    |
| l   | Move right |
| q   | Quit       |
